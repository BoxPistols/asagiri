# Changelog

All notable changes to Asagiri CSS Framework will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2024-01-15

### Added
- Modern CSS Grid layout system with 12-column responsive grid
- Fluid typography using CSS `clamp()` function
- Comprehensive utility classes (600+ TypeScript type definitions)
- Dark mode support with CSS custom properties
- Accessible color system meeting WCAG 2.1 AA/AAA standards
- Component library: buttons, cards, forms, navigation, etc.
- SCSS architecture with modular design tokens
- TypeScript type definitions for all utility classes
- ESM and CommonJS module support

### Features
- CSS Grid-based layout (not Flexbox legacy)
- Mobile-first responsive design
- Japanese typography optimization
- Print styles
- Reduced motion support for accessibility

### Documentation
- Getting Started guide
- Component showcase
- Dark mode implementation guide
- Migration guide for v2.x

## [1.0.0] - 2023-06-01

### Added
- Initial release of Asagiri CSS Framework
- Basic utility classes
- Simple grid system
- Core component styles
