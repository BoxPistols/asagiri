Alert コンポーネント

重要な情報をユーザーに伝えるためのアラート・通知コンポーネント

← ドキュメントに戻る

基本的な使い方

以下のコードをコピーして、そのまま使用できます。

これは基本的なアラートです。重要な情報を表示できます。 
<div class="alert alert-primary">
    これは基本的なアラートです。重要な情報を表示できます。
</div>

カラーバリエーション

8つのカラーバリエーションが用意されています。コピーして用途に合わせて使い分けてください。

Primary - 主要な情報
Secondary - 補足情報
Success - 成功メッセージ
Danger - エラーメッセージ
Warning - 警告メッセージ
Info - お知らせ
Light - 明るい背景
Dark - 暗い背景
<div class="alert alert-primary">Primary - 主要な情報</div>
<div class="alert alert-secondary">Secondary - 補足情報</div>
<div class="alert alert-success">Success - 成功メッセージ</div>
<div class="alert alert-danger">Danger - エラーメッセージ</div>
<div class="alert alert-warning">Warning - 警告メッセージ</div>
<div class="alert alert-info">Info - お知らせ</div>
<div class="alert alert-light">Light - 明るい背景</div>
<div class="alert alert-dark">Dark - 暗い背景</div>

閉じるボタン付きアラート

alert-dismissibleクラスを追加すると、ユーザーが閉じられるアラートになります。

成功！ データが正常に保存されました。
注意： この操作は取り消せません。 
<div class="alert alert-success alert-dismissible">
    <strong>成功！</strong> データが正常に保存されました。
    <button class="alert-close" onclick="this.parentElement.remove()">×</button>
</div>

リンク付きアラート

alert-linkクラスで、アラート内のリンクを強調できます。

新しいアップデートが利用可能です。 詳細を確認する 
<div class="alert alert-primary">
    新しいアップデートが利用可能です。
    <a href="#" class="alert-link">詳細を確認する</a>
</div>

ダークモード対応

Asagiriのアラートは自動的にダークモードに対応します。右上のボタンでテーマを切り替えて確認してください。

<!-- HTMLタグにdata-theme属性を追加するだけ -->
<html data-theme="dark">

<!-- JavaScriptで切り替え -->
<script>
function toggleTheme() {
    const html = document.documentElement;
    const currentTheme = html.getAttribute('data-theme');
    html.setAttribute('data-theme', currentTheme === 'dark' ? 'light' : 'dark');
}
</script>
