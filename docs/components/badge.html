Available Classes

Class Name Description
.badge Base class (required)
.badge-primary Primary color
.badge-success Success color
.badge-danger Danger color
.badge-pill Pill shape (rounded)
.badge-dot Dot status indicator
.badge-pulse Pulse animation
.badge-outline-* Outline version
.badge-sm / .badge-lg Size adjustment
