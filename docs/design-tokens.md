## üìã Recommendations

### Best Practices

1. **Use Design Tokens**: Always use CSS custom properties instead of hardcoded values
2. **Avoid Inline Styles**: Move inline styles to CSS classes
3. **Maintain Consistency**: Reuse existing tokens before creating new ones
4. **Document New Tokens**: Add comments explaining the purpose of new tokens
5. **Test Dark Mode**: Ensure all tokens work in both light and dark themes

### Token Naming Convention

```
--category-property-variant-state

Examples:
--color-primary           (category: color, property: primary)
--color-text-muted        (category: color, property: text, variant: muted)
--spacing-4              (category: spacing, property: 4 = 1rem)
--font-size-lg           (category: font, property: size, variant: lg)
```
